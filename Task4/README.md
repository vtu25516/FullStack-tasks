# Order Management System

## Task 4: Order Management using Joins

This project demonstrates SQL Joins, Subqueries, and ORDER BY using Node.js and MySQL.

### Tables Used
- Customers
- Products
- Orders

### Features
- Customer Order History using JOIN
- Highest Value Order using Subquery

- Simple UI
