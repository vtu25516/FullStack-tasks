# Task 5 - Transaction-Based Payment Simulation

## 📌 Description
This project simulates an online payment transaction system.  
It demonstrates database transactions using **COMMIT** and **ROLLBACK**.

The system:
- Deducts balance from user account
- Adds amount to merchant account
- Uses COMMIT on success
- Uses ROLLBACK on failure

Real-time usage: Banking apps, UPI, Wallet payments, E-commerce checkout.

---

## 🛠 Tech Stack
- Node.js
- Express.js
- MySQL
- HTML (UI Form)
